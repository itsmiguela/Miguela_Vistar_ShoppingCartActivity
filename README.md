Enhanced Shopping Cart System

A console-based shopping cart system built in C#, designed to simulate a real-world checkout process with cart management, stock control, and receipt generation.

Overview

This project demonstrates how core programming concepts can be applied to a practical system.
It focuses on building a structured and reliable shopping experience using simple tools like arrays, loops, and validation.

Key Features
    Product Browsing
        View and search products
        Categorized items for better organization
        
    Cart Management
        Add, update, remove, and clear items
        Prevent duplicate entries
        Maintain accurate stock levels


    Checkout System
        Generates a detailed receipt
        Applies automatic discounts (10% for ₱5000+)
        Calculates payment and change

        
    Stock Control
        Updates inventory in real-time
        Displays low-stock alerts (≤ 5 items)

    
    Order History
       Stores completed transactions
       Displays receipt number, date, and total

        
    Input Validation
       Prevents invalid inputs using safe parsing
       Ensures stable program flow
